# Portfolio images structure

Déposez vos images ici par **service** puis **projet**.

## Structure recommandée

- `/public/images/portfolio/mariage/ninon-alexandre/`
- `/public/images/portfolio/mariage/sokona-julien/`
- `/public/images/portfolio/mariage/karim-ines/`
- `/public/images/portfolio/corporate/hotel-dali/`

## Convention de nommage

`<projet>-<index>.jpg`

Exemples:
- `ninon-alexandre-01.jpg`
- `hotel-dali-01.jpg`

## Bonnes pratiques

- images en `kebab-case`
- cover dédiée (`...-cover.jpg`) si besoin
- garder la cohérence entre nom du projet dans le site et nom du dossier
