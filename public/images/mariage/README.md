# Photos mariage (assets)

Déposez ici vos photos de mariage pour la page `/mariage/`.

## Fichiers utilisés actuellement par la page

La page `/mariage/` utilise directement les photos présentes dans ce dossier.

### Utilisés (démo)

- `mariage-01.jpg` … `mariage-18.jpg`

## Notes

- Chemins publics : `/images/mariage/<nom-fichier>.jpg`
- Formats recommandés : `.jpg` (ou `.webp`)
- Évitez les espaces dans les noms de fichiers.

> Ce dossier est versionné (utile pour GitHub Pages / preview statique).
