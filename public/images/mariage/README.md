# Photos mariage (assets)

Déposez ici vos photos de mariage pour la page `/mariage/`.

- Chemins publics : `/images/mariage/<nom-fichier>.jpg`
- Formats recommandés : `.jpg` (ou `.webp`)
- Nommage conseillé : `mariage-01.jpg`, `mariage-02.jpg`, etc.

> Note : ce dossier est versionné (utile pour GitHub Pages / preview statique).
